## About The Project

To Do List was built for The Odin Project.

-   Create/edit/delete projects.
-   Create/edit/delete to dos within projects.
-   Mark to dos as completed.
-   Save to local storage.
-   Css animations.
-   Fully responsive layout.
-   Uses webPack.
-   Uses Modules.
